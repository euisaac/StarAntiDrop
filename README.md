Anti Drop - Prevent players from dropping items and protect inventories easily.

https://bstats.org/plugin/bukkit/StarAntiDrop

Comandos - Player
/starantidrop ativar - Ativar confirmação de drop
/starantidrop desativar - Desativar confirmação de drop

Comandos - Admin
/starantidrop reload- Recarregar config
/starantidrop adicionar - Adicionar item que está na mão
/starantidrop remover - Remover item que está na mão
/starantidrop atualizar - Verificar atualizações
